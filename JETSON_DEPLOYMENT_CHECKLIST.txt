## ⚠️ Important Notes

### Python 3.8 Constraints
- No `dict[K, V]` syntax - use `Dict[K, V]` from typing
- No `Type | None` syntax - use `Optional[Type]` from typing
- No `match/case` statements (but code doesn't use them)
- Type hints use comment style: `# type: (args) -> return`

### Jetson Specifications
- **CPU**: ARM64 (not x86-64)
- **OS**: Linux (Ubuntu 20.04/22.04)
- **Python**: 3.8 (default on most JetPack versions)
- **CUDA**: 11.8 (Jetson Orin Nano)
- **Memory**: Minimum 4GB RAM for GUI

### Performance Expectations
- **GUI Load Time**: 5-10 seconds
- **Frame Processing**: 5-15 FPS (depends on resolution)
- **Model Inference**: 50-100ms per frame
- **Temperature**: Normal: 45-65°C, Idle: 35-40°C

## 🔧 Troubleshooting

### Installation Issues

**Problem**: PyTorch installation fails
```bash
# Solution: Check internet and retry
pip install torch==1.13.1 torchvision==0.14.1 torchaudio==0.13.1 \
    --index-url https://download.pytorch.org/whl/cu118 -v
```

**Problem**: Virtual environment creation fails
```bash
# Solution: Check Python 3.8 installation
python3.8 --version
which python3.8
```

**Problem**: Serial port not recognized
```bash
# Solution: Check permissions
ls -la /dev/ttyUSB*
id ubuntu  # Check if in 'dialout' group
sudo usermod -a -G dialout ubuntu
```

### Runtime Issues

**Problem**: GUI won't start
```bash
# Solution 1: Set display
export DISPLAY=:0

# Solution 2: Check X11 forwarding (SSH)
ssh -X ubuntu@jetson.local

# Solution 3: Run headless
python vp_runtime.py
```

**Problem**: Out of memory
```bash
# Solution: Stop unnecessary services
sudo systemctl stop bluetooth
sudo systemctl stop avahi-daemon

# Check available memory
free -h
```

**Problem**: Model loading fails
```bash
# Solution: Check model files exist
ls -la dataset/*.pt

# Verify PyTorch
python -c "import torch; print(torch.cuda.is_available())"
```
